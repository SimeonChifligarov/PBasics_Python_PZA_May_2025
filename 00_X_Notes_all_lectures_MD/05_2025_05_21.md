## ğŸ” `while` Loop Mechanics

The `while` loop repeats **as long as a condition is true**.

### ğŸ”¢ Steps:

1. **Initialization**  
2. **Condition check**  
3. **Logic + condition update**

```python
counter = 0                     # 1. initialization
while counter < 5:              # 2. condition
    print(counter)
    counter += 1                # 3. update
```

> âš ï¸ Without the update, the loop may never end!

---

## ğŸ” Example with `break` and `continue`

```python
value = 0
while value < 10:
    value += 1

    if value == 5:
        break        # exits the loop immediately

    if value % 2 == 0:
        continue     # skips to the next loop iteration

    print(value)
```

### Key Notes:
- `break`: stop loop completely  
- `continue`: skip rest of current iteration  
- âœ… Always **update condition before `continue`**

## ğŸ“ `while-else` Construct

Similar to `for-else`, the `else` runs **only if the loop ends without `break`**.

```python
x = 0
while x < 5:
    if x == 3:
        break
    x += 1
else:
    print("Finished without break")
```
